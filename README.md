# TimeDensity Site

Веб-сайт проекта TimeDensity.

## Структура проекта

- `index.html` - Главная страница
- `style.css` - Стили сайта
- `docs/` - Документация
- `faq/` - Часто задаваемые вопросы
- `img/` - Изображения
- `theory/` - Теория
- `visuals/` - Визуализация

## Как использовать

1. Откройте `index.html` в браузере
2. Навигируйте по разделам сайта

## Лицензия

Проект распространяется свободно.
